# Pushkarani Temple Pond Classification System

## Overview

An AI-powered computer vision system for automated classification and analysis of Indian temple ponds (Pushkaranis).

## System Components

- **Backend**: API server with machine learning model integration
- **Frontend**: Web-based user interface  
- **Models**: Multiple deep learning architectures for classification
- **Infrastructure**: Docker-based deployment

## Technical Stack

- Backend: Python with Flask framework
- Frontend: React with modern JavaScript
- ML Framework: TensorFlow/Keras
- Deployment: Docker & Docker Compose

## Installation

### Requirements
- Python 3.9+
- Node.js 16+
- Docker & Docker Compose
- 8GB RAM minimum

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```

### Docker Deployment
```bash
docker-compose up -d
```
